## Getting Started

1. Clone or fork the repo
```sh
git clone https://github.com/cosmoart/jobzilla
```
2. Changue to source directory
```sh
cd source
```
2. Add your own API key in env.local like
```sh
INFOJOBS_TOKEN=XXXXXXXXXXXXXXXXXX
```
3. Install NPM packages
```sh
npm install
```
4. Run the project
```sh
npm run dev
```
